Configuración básica
______________________

[LINUX]Ingresar al sistema:
mysql -h localhost -u root -p

Crear Base de Datos:
CREATE DATABASE mupi;

Crear usuario:
CREATE USER 'mupi'@'localhost' IDENTIFIED BY 'ECOMUPIS';

Dar todos los permisos:
GRANT ALL ON *.* TO 'mupi'@'localhost';

Otros comandos utiles
______________________

Eliminar al usuario:
DROP USER 'mupi'@'localhost';
